Exercício 5.1 - JavaEscripito

Texto padrão do nosso site

-----

Trybe

  1. Um - Crie uma função que mude o texto na tag <P> para uma descrição de como você se vê daqui a 2 anos. (Não gaste tempo pensando no texto e sim realizando o exercício)
  2. Dois - Crie uma função que mude a cor do quadrado amarelo para o verde da Trybe (rgb(76,164,109)).
  3. Tres - Crie uma função que mude a cor do quadrado vermelho para branco.
  4. Quatro - Crie uma função que corrija o texto da tag <h1>.
  5. Cinco - Crie uma função que modifique todo o texto da tag <P> para maiúsculo.
  6. Crie uma função que exiba o conteúdo de todas as tags <P> no console.
